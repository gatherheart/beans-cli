# Claude Instructions

## Development Guidelines

1. **Keep Documentation in Sync**: When making code changes, always update related documentation files. This includes:
   - `README.md` - Update if features, usage, or configuration changes
   - `docs/prd/*.md` - Update the relevant feature document
   - Code comments - Update inline documentation

2. **Documentation Locations**:
   - Feature specifications: `docs/prd/`
   - Development guide: `docs/DEVELOPMENT.md`
   - User documentation: `README.md`

3. **Before Committing**: Ensure all related documentation reflects the current implementation.
