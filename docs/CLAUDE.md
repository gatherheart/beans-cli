# Documentation Directory

## Overview

This directory contains all documentation for the Beans Agent project.

## Structure

```
docs/
├── CLAUDE.md           # This file
├── architecture/       # Architecture documentation
│   └── overview.md     # System architecture overview
└── prd/               # Product requirement documents
    └── *.md           # Feature specifications
```

## Guidelines

1. **Architecture docs**: High-level system design, component interactions
2. **PRD docs**: Feature specifications, requirements, user stories
3. **Keep updated**: Update docs when code changes
4. **Reference gemini-cli**: Check `../gemini-cli/docs` for patterns

## Adding New Documentation

1. For architecture changes: Add to `architecture/`
2. For new features: Add PRD to `prd/`
3. Update README.md for user-facing changes
